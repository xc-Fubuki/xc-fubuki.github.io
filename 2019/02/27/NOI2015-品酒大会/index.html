NOI2015-品酒大会
发表于 2019-02-27| 更新于 2019-12-19
阅读量:

品酒大会的后缀数组+单调栈解法~

n^2暴力: 40pts

考虑钦定后缀i，枚举另一后缀j，利用height数组求出两者LCP，然后统计答案。

n*max{height}暴力: 50pts

利用height求LCP：min{height[rank[i] + 1 .. rank[j]]}

我们发现：[rank[i] + 1, rank[j]]中的最小值决定LCP长度（废话）

我们用nxt[i]表示在height中i右边第一个小于height[i]的位置。显然LCP(i, i + 1 .. nxt[i] - 1) = height[i]，这一部分对答案的贡献可以一起统计（第二问需要用到st表）。

那么我们用单调栈求出nxt[]，统计答案后执行i = nxt[i]，期望得分50。

一点小优化

我们发现执行i = nxt[i]相当于再钦点一次nxt[i]，于是我们用前缀和累计贡献的倍数，同时维护前缀max即可。

时间复杂度：O(n log n)，期望得分100。

还有一点细节就不说了，请自行思考或查看代码。
