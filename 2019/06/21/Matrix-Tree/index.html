Matrix-Tree 矩阵树定理
发表于 2019-06-21| 更新于 2019-12-19
阅读量:

只是简单地记录一下。

对于一张n个点的图求其生成树个数，构造其邻接矩阵。即 $a_{i,j}=a_{j,i}=边(i,j)是否存在$ 。若有重边且重边需要计入方案数，则 $a_{i,j}$ 为 $i$ 与 $j$ 之间边的数量。 $a_{i,i}=-degree(i)$ 。该矩阵的 $n-1$ 阶主子式就是答案。

推广一下，如果边有存在的概率（或者叫边权），那么只要把 $a_{i,j}$ 改成这条边存在的概率， $a_{i, i}$ 改成 $-\sum\limits_{j\neq i}p(i,j)$ 即可。 $p(i,j)$ 就是边 $(i,j)$ 存在的概率。这样求得的是 $\sum\limits_{T}\prod\limits_{e}p_e$ 。

就酱。
