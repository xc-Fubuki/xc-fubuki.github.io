NOI2010·能量采集

据大佬们说，这题就是求$(\sum\limits_{i=1}^{n}\sum\limits_{j=1}^{m}gcd(i, j) *2 - nm)$

那么只要求 $\sum\limits_{i=1}^{n}\sum\limits_{j=1}^{m}gcd(i, j)$ 就好了0v0

$\sum\limits_{i=1}^{n}\sum\limits_{j=1}^{m}gcd(i, j)$

枚举$gcd$：

$\sum\limits_{k=1}^{n}k*\sum\limits_{i=1}^{n}\sum\limits_{J=1}^{m}[gcd(i,j)=k]$

老套路拎出$k$：

$\sum\limits_{k=1}^{n}k*\sum\limits_{i=1}^{\lfloor n/k\rfloor}\sum\limits_{J=1}^{\lfloor m/k\rfloor}[gcd(i,j)=1]$

$gcd$ 改成 $\mu$：

$\sum\limits_{k=1}^{n}k*\sum\limits_{i=1}^{\lfloor n/k\rfloor}\sum\limits_{j=1}^{\lfloor m/k\rfloor}\sum\limits_{d|gcd(i,j)}\mu(d)$

把$d$ 拎到外面

$\sum\limits_{k=1}^{n}k*\sum\limits_{d=1}^{\lfloor n/k\rfloor}\lfloor \frac{n}{kd}\rfloor \lfloor \frac{m}{kd}\rfloor \mu(d)$

最后把 $\mu​$ 筛出来，再用这个式子弄一下就好了

可以证明其时间复杂度为O(n log n)

SDOI2015·约数个数和

为什么不用标准的莫比乌斯反演来弄嘞？我就是不会，你打我呀

还是用$\mu$ 的神奇性质来搞QvQ

$\sum\limits_{i=1}^{n}\sum\limits_{j=1}^{m}d(i*j)$

玄学转换，证明略：

$\sum\limits_{i=1}^{n}\sum\limits_{j=1}^{m}\sum\limits_{x|i}\sum\limits_{y|i}[gcd(x, y)=1]$

枚举$x$和$y$ :

$\sum\limits_{x=1}^{n}\sum\limits_{y=1}^{m}\lfloor\frac{n}{x}\rfloor\lfloor\frac{m}{y}\rfloor[gcd(x, y)=1]$

换成$\mu$ ：

$\sum\limits_{x=1}^{n}\sum\limits_{y=1}^{m}\sum\limits_{d|gcd(x, y)}\mu(d)\lfloor\frac{n}{x}\rfloor\lfloor\frac{m}{y}\rfloor$

换成枚举$d$ ：

$\sum\limits_{d=1}^{n}\mu(d)\sum\limits_{x=1}^{\lfloor n/d\rfloor}\sum\limits_{y=1}^{\lfloor m/d\rfloor}\lfloor\frac{n}{xd}\rfloor\lfloor \frac{m}{yd}\rfloor$

稍微挪下位子：

$\sum\limits_{d=1}^{n}\mu(d)\sum\limits_{x=1}^{\lfloor n/d\rfloor}\lfloor\frac{n}{xd}\rfloor\sum\limits_{y=1}^{\lfloor m/d\rfloor}\lfloor \frac{m}{yd}\rfloor$

然后就能用这个式子暴力计算整除分块了poi

总时间复杂度$\text{O}(T\sqrt{n})$

神TM简单的数学题

求：$\sum\limits_{i=1}^{n}\sum\limits_{j=1}^{n}i·j·gcd(i, j)​$

神TM简单的数学题

$\sum\limits_{i=1}^{n}\sum\limits_{j=1}^{n}i·j·gcd(i, j)$

$\sum\limits_{k=1}^{n}\sum\limits_{i=1}^{n}\sum\limits_{j=1}^{n}i·j·k·[gcd(i, j) =k]$

$\sum\limits_{k=1}^{n}\sum\limits_{i=1}^{\lfloor n/k\rfloor}\sum\limits_{j=1}^{\lfloor n / k\rfloor}i·j·k^3·[gcd(i, j)=1]$

$\sum\limits_{k=1}^{n}\sum\limits_{i=1}^{\lfloor n / k\rfloor}\sum\limits_{j=1}^{\lfloor n / k\rfloor}i·j·k^3·\sum\limits_{d|gcd(i, j)}\mu(d)$

$sum(n)=\frac{n(n+1)}{2}$

$\sum\limits_{d=1}^{n}d^2·\mu(d)\sum\limits_{k=1}^{\lfloor n/d\rfloor}d^3·sum^2(\lfloor\frac{n}{kd}\rfloor)$

$T=kd$

$\sum\limits_{T=1}^{n}sum^2(\lfloor\frac{n}{T}\rfloor)\sum\limits_{k|T}k^2\mu(k)·(\frac{T}{k})^3$

$\sum\limits_{T=1}^{n}sum^2(\lfloor\frac{n}{T}\rfloor)·T^2\sum\limits_{k|T}k·\mu(\frac{T}{k})$

$f(i)=i^2\sum\limits_{k|i}k·\mu(\frac{T}{k})$

$=i^2·(id*\mu)(i) $

$=i^2·\varphi(i)$

$S(n)=\sum\limits_{i=1}^{n}f(i)$

$=（\text{杜教筛推导，不在本文涉及范围内}）$

$=sum^2(n)-\sum\limits_{i=2}^{n}i^2·S(\lfloor\frac{n}{i}\rfloor)$

然后结束，整除分块+杜教筛即可。
