仍然用树剖+bitset，处理出每一个人的bitset后，尝试二分答案。

若当前二分得到的答案为mid，把每个人向她可以获取的颜色连边，并把这个人复制mid份。

然后跑二分图最大匹配。

恭喜你，你已经能切掉此题了

显然这样复杂度爆炸。

使用Hall定理：定理内容自行百度qwq

我们考虑人的一个非空子集（这些人复制了一堆）U，她们连出去的边对应点集V，则存在完美匹配的条件为|U|<=|V|。

显然我们不能枚举所有子集。

若当前答案为k，某一子集S中存在cnt < k个点复制自同一个人，那么从这个人复制k个点到S中所取得的限制条件一定更强（即|U|变大，|V|不变）。

也就是说，只需枚举人的子集。

这时，我们发现答案根本不需要二分：令当前枚举到人的子集大小为size，她们一共能获取至多tot种颜色，那么答案k必须满足size * k <= tot，把size丢到右边去：k <= tot / size

时间复杂度:$O(q·log^2 n·\frac{m}{64}+2^c·q·\frac{m}{64})$

期望得分：100， 实际得分：100
