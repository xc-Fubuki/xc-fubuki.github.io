Sol-1

线性基，然而我不会qwq

Sol-2

打表找规律（打出i = 0 to 24, [L, L + i]的答案）

我惊喜的发现：答案是2的整次幂（废话。

显然，每个二进制位对答案的贡献独立，那么把它们分开考虑；

用vis[i]标记第i位是否产生了贡献；

由于我讲不出道理，所以接下来简述流程：

  • 标记L的最左边二进制位

  • long long x = L;

  • 从0开始枚举二进制位，这一位上的值为1时弹出。若x | (1 << i) <= R，则第i位可以产生贡献，标记。

  • x += lowbit(x);

  • 若x <= R，则x的最右边二进制位能产生贡献，标记。若x > R, 退出循环。

然后就A了此题qwq

Sol-3

若R - L > 1，则第0个二进制位一定能产生贡献，递归处理[L >> 1, R >> 1]，直到R - L < 1。
