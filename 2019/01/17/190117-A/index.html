n^2暴力

n^2的dp很显然qwq

设f[i]为按t排序后把前i个人送到并返回大厅的最短用时。

转移：f[i] = min{ max(a[i].t, f[j - 1]) + a[j].x * 2 }

暴力的优化

我们发现，对于两个人i和j，若a[j].x <= a[i].x && a[j].t <= a[i].t，则j与i一起走最优（也就是说能忽略a[j]

按t排序，把没用的人杀掉，这样就可以拿到96

优化2.0：奇技淫巧

我们发现：这个dp有近似的决策单调

那么单调队列单调栈什么的瞎搞一下，优化下常数就行了。

然而，已知最高乱搞的分为99

优化3.0： 正经的优化

f[i] = min{ max(a[i].t, f[j - 1]) + a[j].x * 2 }

考虑f[j - 1] <= a[i].t：前缀和形式处理即可；

否则：单调队列维护即可。
